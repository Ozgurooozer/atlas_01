### 🔵 FAZ 1: 2.5D TEMEL (Hafta 1)
**Hedef**: Isometric foundation
**Kalite**: %40 → %60

#### Gün 1-2: Isometric Projection System
```
engine/renderer/isometric.py
├── IsometricProjection
│   ├── world_to_screen()       # World → Screen koordinat
│   ├── screen_to_world()       # Screen → World koordinat  
│   ├── get_depth_sort_key()    # Z-sorting için key
│   └── apply_height_offset()   # Yükseklik hesaplama
└── IsometricCamera
    ├── set_view_angle()        # 45°, 30°, vs.
    ├── set_zoom_level()        # Zoom in/out
    └── follow_target()         # Smooth follow
```

**Test**: `tests/renderer/test_isometric.py`
- 100+ test case
- Coordinate conversion accuracy
- Depth sorting correctness

#### Gün 3-4: Height/Depth System
```
engine/renderer/height_sprite.py
├── HeightSprite
│   ├── base_position: Vec2      # X,Y floor position
│   ├── height: float           # Z elevation (0-100)
│   ├── depth_sort: float       # Y + Z * 0.5
│   └── render_with_height()    # Height offset uygula
└── HeightMap
    ├── load_heightmap()        # Grayscale image
    ├── get_height_at()         # X,Y → height
    └── apply_to_sprite()       # Sprite'a uygula
```

**Features**:
- Sprite elevation (zıplama, platform)
- Depth-based occlusion
- Height-based shadow offset

#### Gün 5-7: Depth Sorting & Layering
```
engine/renderer/layer_manager.py
├── LayerManager
│   ├── LAYERS = {
│   │   'background': -100,
│   │   'floor': 0,
│   │   'shadow': 1,
│   │   'characters': 10,
│   │   'fx_back': 20,
│   │   'fx_front': 30,
│   │   'ui': 100
│   │ }
│   ├── add_to_layer()          # Sprite layer'a ekle
│   ├── sort_layer()            # Depth sort
│   └── render_layers()         # Layer sırasıyla render
└── DepthSortKey
    ├── y_position              # Y koordinat
    ├── height                  # Z elevation
    └── custom_priority         # Manuel override
```

**Milestone 1**: 
- ✅ Isometric projection çalışıyor
- ✅ Height system aktif
- ✅ Depth sorting doğru
- **Kalite**: %60

### 🟢 FAZ 2: GELİŞMİŞ IŞIKLANDIRMA (Hafta 2)
**Hedef**: Hades-level lighting
**Kalite**: %60 → %75

#### Gün 8-10: 2D Normal Maps
```
engine/renderer/normal_lighting.py
├── NormalMapTexture
│   ├── diffuse: Texture        # Renk (RGB)
│   ├── normal: Texture         # Normal map (RGB)
│   └── specular: Texture       # Specular (A)
├── NormalMapShader
│   ├── calculate_lighting()    # Dot product
│   ├── apply_specular()        # Parlama
│   └── blend_normals()         # Multi-light
└── NormalMapRenderer
    ├── bind_normal_map()       # Shader'a bind
    ├── set_light_positions()   # 8 light max
    └── render_sprite()         # Final composite
```

**Tools**:
- SpriteDLight (normal map generator)
- Laigter (free alternative)

**Test**: `tests/renderer/test_normal_lighting.py`

#### Gün 11-13: Soft Shadows
```
engine/renderer/soft_shadows.py
├── ShadowCaster
│   ├── cast_shadow()           # Shadow raycast
│   ├── soft_shadow_kernel()    # Gaussian blur
│   └── self_shadow()           # Self-occlusion
├── ShadowMap
│   ├── generate_shadow_map()   # 2D shadow texture
│   ├── blur_shadows()          # Soft edge
│   └── composite_shadows()     # Final shadow
└── ShadowLight
    ├── position: Vec3          # X,Y,Height
    ├── radius: float           # Light range
    └── intensity: float        # Shadow darkness
```

**Features**:
- Soft penumbra (yumuşak kenar)
- Self-shadowing
- Height-based shadow offset
- Multiple shadow casters
